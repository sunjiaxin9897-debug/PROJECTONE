Teacher / Project Notes

Text and graphics: child-friendly cards, large icons, simple vocabulary, colorful layout, and CSS-made scenes.

Audio and video: intro video, optional how-to-play video, BGM, reward music, and sound effects are connected through relative asset paths. The page still works even if a media file is missing.

Animation: CSS floating leaves, mascot movement, card flipping, clouds, item pulse, progress bar, and badge pop animation.

Interaction: name input, badge selection, mascot tips, flip cards, promise selection, park clean-up game, drag/click sorting game, quiz, score, stars, feedback, and reward certificate.

Gitee deployment: upload this file and the assets folder to the repository root. Keep all media paths relative.
